Taco Bell Path

This is a project I worked on with a few of my friends for our CS 225(Data Structures) final project. The goal of this project was to find the shortest route from Champaign, IL to Union Station in Chicago that went through the most number of Taco Bells. We used a graph structure to store the data, and Dijkstra's algorithm to determine the shortest path. Albeit a silly project, it was the first time I worked in a group to accomolish a common objective. I learned a ton about peer programming, version control, and the importance of testing. My contribution included web scraping to aggregate a list of all Taco Bells and their addresses in Illinois into a csv file format. Additionally, I wrote test cases for our Dijkstra's algorithm implementation. The code is available on github here: Taco Bell Path Github
